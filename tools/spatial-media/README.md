# 360 Video Metadata Tool

`Spatial Media Metadata Injector.exe` is included as a manual compatibility tool for existing videos.

Particle Model Studio's **360° MP4** export injects mono equirectangular spherical-video metadata automatically, so this utility is normally not required. Use the utility only when adding 360 metadata to an MP4 created elsewhere.

The utility comes from Google's `spatial-media` project and is distributed under the Apache License 2.0. See `LICENSE-spatial-media.txt`.
