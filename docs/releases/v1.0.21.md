# Particle Model Studio v1.0.21

这是一次从“功能很多”转向“少量参数快速出片”的体验升级，同时汇总了 v1.0.18—v1.0.21 的渲染、图谱和镜头改进。

## 重点更新

- 新增三种 Creator 一键风格：流体丝带、风化烟尘、能量爆散。
- 消散改为连续侵蚀前沿、束状拖丝、边缘碎片和受控雾化。
- Creator 默认关闭 GPU Particle Feedback，速度为 0 时完全静止。
- 每个模型维护独立流动相位；选择、取消选择或切换模型不会改变运动。
- 时间轴、相机预览、静帧和视频导出使用确定性速度积分。
- 新增可见风格卡和“已静止 / 流动中”实时状态。
- 新增正式应用图标，替换 Electron 默认图标并统一任务栏、资源管理器与仓库视觉。
- 中文节点、端口和参数；中键平移、左键选择并拖拽节点。
- 自定义 Graph 继续支持 Emitter、Birth/Life、Force、Return、Attractor、Collision、Trail 与 GPU Feedback。
- 多层深度辉光独立控制半径与曝光，低强度过渡更加连续。
- 景深升级为薄透镜 CoC 与多边形光圈散景，并在主视口、相机预览和导出保持一致。
- 改进近景粒子密度，镜头靠近时不再出现模型采样消失。

## 兼容性

- `.pms` schema 继续保持版本 1，新字段均为可选字段。
- 旧 Creator 工程自动迁移到“流体丝带”，保留原有进度、速度、颜色和高级参数。
- 缺少新字段的旧自定义 Graph 继续使用旧算法，不覆盖手工图。
- Windows x64，解压后运行 `Particle Model Studio.exe`。

## 验证

- 59 项图谱、资源、反馈、景深与粒子流单元测试通过。
- 开发版和打包版 Electron/WebGL 烟测通过，无 Shader 编译错误。
- 双模型选择切换平均像素差为 0.0053，变化像素比例为 0.0069%。
- `Particle-Recovery.pms` 中实际加载的 21 个模型速度均为 0，等待后相位仍全部为 0。
- 同一工程近景可见粒子量约为远景的 2.568 倍，不再靠近后消失。
