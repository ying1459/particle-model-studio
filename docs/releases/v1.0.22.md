# Particle Model Studio v1.0.22

这是一个针对 Creator 工作流的模型逸散入口补丁。

## 修复

- 点击右侧“逸散”属性页时自动切换到模型逸散渲染模式，不再只显示参数却继续渲染普通粒子。
- 点击“粒子”“生长”或“消散”属性页时自动回到模型粒子模式。
- 顶部模式按钮、右侧属性页和当前渲染模式保持同步。
- “逸散”分类图标由容易误解的“发”调整为“逸”，并增加操作提示。
- 保留旧工程中的 `emissionEnabled`、数量、强度、距离、速度、风场、粒子材质和局部破碎参数。

## 验证

- `Particle-Recovery.pms` 中逸散开关及全部主要参数仍然存在。
- 59 项核心测试通过。
- Electron 完整界面测试通过，并验证逸散页、模式按钮、渲染状态和返回粒子模式的双向联动。
